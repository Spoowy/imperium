@imperium/auth

Auth

Model class for authentication. Uses a Mongo roles collection and has an opinionated user object.

new Auth(connection, ctx)

ParamTypeDescription
connectionobjectThe Mongo connection.
ctxobjectThe Context object that this model belongs to.

Creates a new Auth model.

auth.defaultAuth()

Returns a default (blank) authentication object (for server)

Returns: Map - An Immutable map of a blank authentication object.

auth.buildAuthFromJwt(decodedJWT)

Param
decodedJWT

Builds an authentication object from a decoded JWT

Returns: Promise.<Map> - An Immutable map of the authentication object created from decoded JWT data.

auth.serializeAuth(auth)

ParamTypeDescription
authMapThe Immutable Map that will be serialized.

Takes in an authentication object and serializes it for transport to the client.

Returns: Promise.<Map> - An Immutable map that can be serialized using Transit Immutable.

auth.signIn(email, password)

ParamTypeDescription
emailstringThe email to sign in with.
passwordstring \objectThe password string/object to log in with.

Attempts the sign in process.

auth.generateJwt(payload, options)

ParamDescription
payloadOptional JWT claims
optionsOptional JWT options

Generate a JWT for the currently logged in user.

auth.encryptPassword(password)

Param
password

Encrypts a password

Param
roles

Gets the array of permissions from an array of roles

Param
name
permissions

Create a new role

Param

signOut(apolloClient, history)

ParamDescription
authMust be an immutable auth object
